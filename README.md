# FYNO Client

1. Clone the repo
```sh
git clone https://github.com/vincent0426/fyno-client.git
```
2. Go into the repo
```sh
cd fyno-client
```
3. Install dependencies
```sh
yarn install
```
4. Add .env file and ask for the content
```sh
cp .env.example .env
```
5. Start the dev server
```sh
yarn dev
```
6. Open http://localhost:5173
