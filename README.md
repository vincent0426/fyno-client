# Find your next owner

1. Clone the repo
```sh
git clone https://github.com/vincent0426/find-the-next-owner-client.git
```
2. Go into the repo
```sh
cd find-the-next-owner-client
```
3. Install dependencies
```sh
yarn install
```
4. Start the dev server
```sh
yarn dev
```
5. Open http://localhost:5173
